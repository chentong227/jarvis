## 0. 衡是什么 / 与锚化的分界

- **衡 = 判断 faculty。** 想自由涌念(发散),衡对着墙判断、取舍、解冲突、记代价(收敛)。
- **分界:** 锚化造**墙**(言出必行/灵魂层是边界);衡在墙上、墙间**权衡**(选哪墙守)。墙是名词,衡是动词。
- **横切器官:** 衡像锚一样跨器官 —— 在 **想**(后台慎思,慢)+ **口**(现场回复,快,TTFT<5s)里都发生;由 **体** 的锚冲突区召唤。

> 一句话定位:前三元架构(说/想/体)管"谁说、谁想、被什么驱动";**衡管"在边界张力上怎么收敛成一个选择"。**

---

## 1. 想 / 衡 = 发散 / 收敛 [理念源 §9]

| | 想(识) | 衡 |
|---|---|---|
| 方向 | 发散 divergent | 收敛 convergent |
| 做什么 | 体 delta 召唤 → 自由涌念、联想、铺可能性(墙内软) | 对着墙判断/取舍/选/解冲突 |
| 层 | 软(性格在此自由移动) | 硬冲突在此裁决 + 软选择在此收口 |

这对是任何认知 act 的两半(生成 vs 评判)。从前我们只有"想",收敛靠零散的 cooldown/saturation —— 衡把收敛**显式化**成一个 faculty。

---

## 2. 衡的三态(精确化 Layer 1 的"放电 or 休息") [接 commit a650323]

| 态 | 衡的定义 | 闭环("想完了") |
|---|---|---|
| **放电 discharge** | 解一个**边界张力**(选哪墙守 / want 顶墙该不该越)→ 收敛成一个选择(+越墙记代价) | 张力解了 → 停 |
| **休息 rest** | 无张力(诸墙不冲、无紧迫 want)→ 安静(头等公民) | 无张力 → 歇 |
| **filler(杀)** | 内部深处空转,离任何墙都远、无 want | —— Layer 1 已做雏形 |

**关键:锚给了"想完了"的停**——撞到一堵墙 = 这事不做、到此为止、没什么可想了(理念源 §9)。无锚时反刍停不下来,正因没墙可答它。

---

## 3. 衡的冲突裁决:自我在此锻造 [理念源 §5;依赖锚化]

- 两墙同时撑不住 → **无固定等级**(Q-a,拆"真相>取悦")→ **逐案选哪墙守 + 把越墙登记为伤**。
- 伤 → 河床高值记忆 → 塑后续可塑性(理念源 §3-4b)。
- **那一串逐案选择的样子 = 性格。** 优化器挑高分转头就忘(无伤);衡破墙、知道破了、带着伤。
- 经典冲突面:**诚实(锚#1)vs 善意(锚#2)**。Jarvis 怎么 navigate 它 = 它的性格维度。

## 4. 衡横切器官:想(后台) + 口(现场) + 体(召唤)

| 器官 | 衡在这里的形态 |
|---|---|
| **想**(后台) | 慎思:慢、可多轮逼近一个冲突的解 |
| **口**(现场) | TTFT<5s 当场权衡(诚实 vs 善意),快;衡服务于口的回复生成 |
| **体**(召唤源) | 提供"锚冲突区"作为最高势能召唤信号(两定点 tension = 该想了) |

## 6. 风险 / 红线

- **别把衡做成又一个梯度/优化器**:衡是"对着墙收敛成选择",不是"最大化某分"。若衡变成打分排序,就退回函数(理念源 §10)。
- **记代价不可被优化掉**:伤必须真留痕(河床),否则就是优化器"挑高分转头忘",勇气/性格无从锻造。
- **站立限制(理念源 §10):** 衡能让收敛"诚实"(解/歇/记伤),但**给不出 telos**;它是清场+收敛,不制造 inside。
